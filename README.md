# Projeto-Log-Analyzer
Este é um projeto pessoal. Programei um analisador de logs usando Python.
Caso queira utilizar ou testar, apenas baixar e usar, todas as bibliotecas já vem com python.
Utilizei como teste o sample de log de um servidor Apache do repositório https://github.com/logpai/loghub/tree/master/Apache
