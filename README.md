# Projeto-Log-Analyzer
Este é um projeto pessoal. Programei um analisador de logs usando Python.
Iniciei este projeto com o objetivo de aprimorar minhas habilidades em python e que inclua meus conhecimentos em segurança cibernética que adquiri na faculdade.
Caso queira utilizar ou testar, apenas baixar e usar, todas as bibliotecas já vem com python.
Utilizei como teste o sample de log de um servidor Apache do repositório https://github.com/logpai/loghub/tree/master/Apache

# English

This is just a personal project. I programmed a log analyzer using Python.
I started this project to improve my skills in Python while also including my knowledge in cyber security that I learned in university.
If you want to use or test it, just download and use it, all the libraries come pre-installed with python.
I tested my code using an Apache log sample from the repository https://github.com/logpai/loghub/tree/master/Apache
